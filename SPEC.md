## 新機能（v1.4.0）

### M8: ツールチップ・入力ガイダンス・ヘルプテキスト充実
- **? アイコン + ホバーツールチップ**: 各質問の右に「?」ボタンを配置
  - マウスホバー時に詳細説明・入力例をポップアップ表示
  - モバイル対応：固定ツールチップで画面上に表示
- **入力フォーム説明の充実**:
  - 目標月額入力欄に詳細なガイダンステキストを追加
  - 「目標月額の設定について」説明パネルを追加
  - 入力例（3000円、2500円、1500円）を目に見える形で表示
- **初期ガイドの強化**:
  - 「このツールの使い方」セクションを上部に追加
  - 5つのステップを絵文字で視覚化
  - ? アイコンの説明をここで先制説明
- **質問ごとの詳細ガイダンス**:
  - 全5問各々に「help」（説明）と「example」（具体例）を設定
  - 例1: 「現在のキャリア」→ マイページまたはスマホ設定で確認する方法を説明
  - 例2: 「データ使用量」→ 動画視聴時は10GB以上など具体例を提示
  - 例3: 「優先事項」→ 「毎月3,000円以下→『とにかく安さ重視』」など判断基準を提示
- **Aria属性・Title属性追加**:
  - input要素に `aria-label="目標月額（円）"` を追加
  - title属性で入力単位・説明を追加

**実装詳細**:
- questions オブジェクトに `help` / `example` フィールドを追加
- renderQuestion() 関数で ツールチップHTMLを動的生成
- CSS で `.group:hover .group-hover:block` でツールチップ表示制御
- モバイル対応：@media (max-width: 640px) で固定ツールチップに切り替え

**効果**:
- ユーザーが入力項目の意味を即座に理解できる
- 入力例で迷いなく選択肢を選べる
- 新規ユーザーの離脱防止
- アクセシビリティ向上（aria-label / title）

## 新機能（v1.3.0）

### M3: データ保存・履歴機能（localStorage）
- 診断回答を自動保存し、最大5件の履歴を保持
- 「前回の診断を復元」ボタンで即座に前回結果を表示
- 履歴クリア機能で手動削除可能

### M4: グラフ・スコア可視化（Chart.js）
- Chart.jsで料金比較グラフ（棒グラフ）を描画
- 現在料金（赤）→ 乗り換え後（緑）→ 節約額（青）を視覚化
- レスポンシブ対応でモバイル・PCで自動適応

### M6: 目標設定・達成度トラッキング
- 診断前に「目標月額」を入力可能
- 達成率を0〜100%で可視化（プログレスバー表示）
- 目標値をlocalStorageに保存し、複数診断で比較可能

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **保存データ**:
  - 診断履歴（smartphone_advisor_data）: 最新5件、タイムスタンプ付き
  - 目標値（smartphone_advisor_goal）: 単一の目標月額
- **永続化**: ページ読み込み時に自動復元・復元UI表示
- **クリア**: ブラウザのサイトデータ削除で初期化、または「履歴をクリア」ボタン

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する
